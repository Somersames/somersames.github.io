Spring Cloud Config

使用Spring Config来配合git做一个配置文件管理，需要一个Config的服务端和一个Config的客户端，服务端主要是和git仓库进行一个连接，而config的客户端是连接服务端来刷新配置服务的。
在Spring Cloud Config里面客户端需要使用Spring4.0出现的一个注解@Value配合一起使用
